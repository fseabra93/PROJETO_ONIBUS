Sistema Gerenciador de Transportes Coletivos

O "Sistema Gerenciador de Transportes Coletivos" é um programa de gerenciamento de linhas de ônibus urbanos.


## Instalação

1. Clone o repositório;
2. Abra o terminal e navegue até o diretório do projeto;
3. Execute o comando `npm install`.

## Utilização

1. Execute o comando `npm start` para iniciar o servidor;
2. Abra o navegador e acesse `http://localhost:3000`.

## Contribuição

1. Faça um fork do projeto;
2. Crie uma nova branch com as suas alterações: `git checkout -b minha-branch`;
3. Faça o commit das suas alterações: `git commit -m 'Minha alteração'`;
4. Faça o push para a sua branch: `git push origin minha-branch`;
5. Abra um Pull Request.
